# Plano Gratuito

- 1 contato de emergência
- 1 alerta por dia
- Mensagem enviada a(o) cuidador(a) somente via App (o cuidaddor precisa ter o app instalado para receber os alertas)
- Exibição de propagandas no App

# Plano Vip

- 3 contatos de emergência
- Check-ins ilimitados
- Lembretes ilimitados
- Alertas ilimitados
- Botão de "Estou Mal" - alerta instantâneo
- Mensagem enviada a(o) cuidador(a) via App e SMS/WhatsApp (será enviado em segundo plano, o usuário receberá uma notificação no celular)
- Sem propagandas
